NVDLA

The NVDLA (Nvidia Deep Learning Accelerator) is an alternative to the GPU, and it can be used with minimal change in software. It is not as fast as the GPU but is extremely power efficient. So, if you need your application to run at all times and power consumption is an issue, you should try to use the DLAs.
